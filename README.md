 # Antigravity를 이용한 주식 메모 프로그램


 파일럿 프로젝트입니다. 요건은 다음과 같습니다
 - 직접 코딩을 최소화하고, 최대한 Antigravity를 이용하여 작업한다.
 - CI/CD는 github actions Self-hosted Runner를 이용한다.
 - 배포환경은 Cloud환경으로 진행한다.  
  금전적인 문제로 자체 private Cloud환경을 구축하여 진행한다.  
  ngrok, docker, k8s를 이용하여 차례대로 적용한다.
 - 기능적 목표는 김어준의 방송의 주식 아가방 프로그램에서 애기하는 기능을 최대한 적용한다.
